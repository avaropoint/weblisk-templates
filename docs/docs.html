Introduction

My App is built with Weblisk — a zero-dependency web framework that embraces the platform. No virtual DOM, no bundler, no magic.

Features

  • Islands architecture — JavaScript loads only where needed
  • Reactive signals — fine-grained reactivity without a runtime
  • View transitions — smooth MPA page transitions
  • Service workers — offline support and push notifications
  • Dark mode — built-in theme toggling

Installation

Install the Weblisk CLI globally:

npm i -g weblisk

Quick Start

weblisk new my-app --template docs
cd my-app
weblisk dev
